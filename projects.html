PfSense Router/Firewall

Overview:

I designed and deployed a secure authentication system using smart cards to enhance enterprise login security. This involved setting up a Certificate Authority (CA), issuing digital certificates, and configuring multi-factor authentication (MFA) protocols across client machines. The project emphasized Public Key Infrastructure (PKI) principles and enterprise-grade identity management.

The system was built to simulate a real-world environment where users authenticate using smart cards and PINs, ensuring strong access control and minimizing credential-based attacks.

Complex Challenges Faced:

  • Managing certificate lifecycles, including renewal and revocation processes
  • Ensuring compatibility between smart card readers and diverse client machines
  • Troubleshooting authentication failures and certificate trust issues
  • Balancing usability with strict security requirements
  • Documenting setup procedures for future scalability and training

Ethical Considerations and Actions Taken:

  • Ensured all certificate issuance followed secure identity verification practices
  • Maintained transparency in system limitations and user onboarding procedures
  • Prioritized user privacy and secure handling of authentication data
  • Created clear documentation to support ethical deployment and maintenance

Outcomes:

  • Successfully deployed a smart card-based authentication system with MFA
  • Strengthened understanding of PKI, certificate management, and secure login protocols
  • Demonstrated the effectiveness of hardware-based authentication in enterprise environments
  • Produced reusable documentation for future implementations and training
