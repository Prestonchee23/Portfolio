Mathew Made Construction Internship

Networking, secure remote access, VLAN segmentation, and security documentation.

Overview:

During my internship, I faced and solved a lot of real-world networking challenges. I was responsible for building and troubleshooting a secure remote access system, diagnosing and fixing intermittent internet connectivity issues. I configured VLANs to separate guest and internal traffic, which involved detailed troubleshooting and fine-tuning routing between the switch and firewall.

Besides VLAN setup, I worked directly with the company’s firewall to manage security rules and remote access. I also configured a network-attached storage (NAS) system to support data backup and sharing across departments. To keep everything running smoothly, I set up and optimized a managed Layer 2 switch that supports Power over Ethernet (PoE), ensuring proper traffic flow between devices.

As part of the company’s push toward Cybersecurity Maturity Model Certification (CMMC) Level 1, I helped review system setups to meet compliance requirements and assisted in drafting clear security policies and procedures.

These hands-on experiences gave me a solid understanding of enterprise networking, system configuration, and security best practices. I’m also working on documentation and helping create incident response plans and security policies to further improve the company’s cybersecurity.

Complex Challenges Faced:

  • Building and troubleshooting secure remote access with unreliable internet connections.
  • Configuring VLANs and troubleshooting routing issues between switches and firewalls.
  • Managing firewall rules and remote access settings to keep security tight.
  • Integrating NAS systems to support backup and file sharing.
  • Optimizing Layer 2 switches with PoE to ensure network stability.

Ethical Considerations and Actions Taken:

  • Avoided using real customer data during testing to protect privacy, opting for synthetic data instead.
  • Implemented strict access controls and multi-factor authentication to safeguard sensitive information.
  • Maintained transparency with supervisors about security risks, always following company policies and legal requirements.

Outcomes:

  • Improved network reliability and security across the company.
  • Helped the company move closer to CMMC Level 1 compliance.
  • Gained practical experience balancing technical needs with ethical responsibility.
